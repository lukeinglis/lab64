# Orange Cat

## Character Brief

| Field | Value |
|---|---|
| Animal Type | Cat (domestic shorthair, orange tabby) |
| Visual Identity | Orange tabby stripes, mischievous expression |
| Distinguishing Features | Bold orange and darker orange stripe pattern, wide eyes, expressive ears |
| Target Racer Slot | TBD (owner to confirm; default assumption: Toad) |

## Design Notes

- **Split-screen readability:** The warm orange color is highly visible and distinct from all other characters. Tabby stripes add visual interest even at small sizes.
- **First-pass goal:** A clear orange tabby cat racer. Mischievous and energetic.
- **Silhouette:** Similar slim cat build to the Black Cat but distinguishable by ear angle and expression. Slightly rounder face.
- **Palette:** Orange body, darker orange tabby stripes, yellow/amber eyes, white chest patch, pink nose.
