# Yellow Lab

## Character Brief

| Field | Value |
|---|---|
| Animal Type | Dog (Labrador Retriever) |
| Visual Identity | Golden/yellow coat, friendly expression, floppy ears |
| Distinguishing Features | Warm golden color, broad head, happy open mouth |
| Target Racer Slot | TBD (owner to confirm; default assumption: Luigi) |

## Design Notes

- **Split-screen readability:** The solid golden color is distinct from the Dalmatian's spotted white. Should read as "the yellow one" at a glance.
- **First-pass goal:** A clear yellow Labrador-inspired racer. Friendly and approachable.
- **Silhouette:** Slightly stockier than the Dalmatian, broader head, thicker neck.
- **Palette:** Golden/yellow body, darker golden ears, brown nose/eyes, optional blue collar.
