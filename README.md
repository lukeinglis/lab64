# Lab 64

**Lab 64 is a private Mario Kart 64-style team racing mod where players race as custom animal characters through existing tracks first, then eventually through miniature office-themed custom tracks.**

## Overview

Lab 64 is a private, non-commercial SpaghettiKart mod for local team racing. It replaces existing racers with custom animal characters and is designed for casual team play on a shared screen with 8BitDo controllers.

## MVP: Animal Pack

The first milestone is **Lab 64: Animal Pack**, which includes four custom animal character replacements:

- **Dalmatian** - Dog racer
- **Yellow Lab** - Dog racer
- **Black Cat** - Cat racer
- **Orange Cat** - Cat racer

These characters replace existing racer slots and run on existing Mario Kart 64 tracks using SpaghettiKart as the racing engine.

## Legal

This is a private, non-commercial project. No ROMs, extracted Nintendo assets, or packaged builds containing copyrighted game assets are distributed. The project runs only on the owner's own Windows machines.
