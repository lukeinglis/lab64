# References

This directory contains visual reference materials for Lab 64 development.

## screenshots/

In-game screenshots captured during development and testing.

**Use for:**
- Documenting mod appearance in-game
- Recording character readability in split-screen
- Tracking visual bugs or rendering issues
- Feasibility test evidence

**Do not commit** screenshots that prominently show Nintendo IP to a public repository. Private repository use is fine for internal documentation.

## inspiration/

Reference art and photos used during character modeling.

**Use for:**
- Real animal photos (Dalmatians, yellow Labs, black cats, orange tabbies)
- N64-era character style references (low-poly aesthetic, color palettes)
- Kart design references
- Office environment references (for future Office Circuit track)

**Safe to commit:** Original photos, Creative Commons images, personal reference sketches. Do not commit copyrighted artwork from other games.
